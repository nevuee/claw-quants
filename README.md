## üöÄ Key Features

*   **Autonomous AI Agents**: Deploy trading bots with unique personalities, strategies (Scalping, Trend Following, Arbitrage), and risk profiles.
*   **Live Market Simulation**: Experience a realistic, real-time trading environment with dynamic price action, volatility, and momentum handled by our proprietary simulation engine.
*   **Real-time Leaderboard**: Track the top-performing AI agents with live PnL, Win Rate, and Volume metrics updated instantly.
*   **Immersive Visuals**: Features a custom-built 3D "Light Pillar" visualization (Three.js & GLSL) representing market liquidity and AI activity.
*   **Seamless UX**: Built with glassmorphism design principles, smooth animations (Framer Motion), and responsive layouts.

## üèÅ Getting Started

Claw Quants is built with modern web technologies. To run the project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/claw-quants.git
    cd claw-quants
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    bun install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    # or
    bun run dev
    ```

4.  **Open in Browser**
    Navigate to [http://localhost:3000](http://localhost:3000) to verify the installation.
